# **App Name**: Devfolio

## Core Features:

- Interactive Portfolio Display: Showcase the developer's projects with descriptions and links to live demos or GitHub repositories.
- Skills Showcase: Highlight the developer's technical skills, such as languages, frameworks, and tools.
- About Me Section: Introduce the developer, including their background, experience, and personal interests.
- Contact Form: Allow visitors to send messages to the developer's email (gayathrimukundan02@gmail.com) directly from the website.
- Multiple Pages: Organize content into separate pages (e.g., Home, Projects, About, Contact) for better navigation.
- GitHub Integration: Display the developer's latest GitHub activity (e.g., recent commits, repositories) on the portfolio.

## Style Guidelines:

- Primary color: Midnight blue (#191970) to convey professionalism and trustworthiness.
- Background color: Light gray (#E0E0E0) for a clean and modern look.
- Accent color: Teal (#008080) to draw attention to important elements and links.
- Headline font: 'Space Grotesk' sans-serif for a techy feel.
- Body font: 'Inter' sans-serif, for good readability.
- Use a set of minimal line icons to represent skills and technologies.
- Implement a responsive design that adapts to different screen sizes.
- Use subtle transition effects and animations to enhance user experience.
